## Project Summary

The **Mandalay Citizen Fire-Report and Route System** is a web-based application that helps citizens report fire incidents quickly and accurately. Users can submit a fire report with one click, and their current location is automatically shared with the nearest fire station.

The system provides a real-time dashboard for fire stations, finds the nearest fire station, suggests the best route, and recommends firefighting resources based on the fire severity. It also displays nearby fire hydrants and water sources to support emergency response.

This project aims to reduce response time, improve coordination, and help protect lives and property in Mandalay.
